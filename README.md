# Simple Headless Carousel React

### How to use:

```
<CarouselProvider width={500} total={3}>
  <Carousel>
    <Slide index={0}>
      <Img />
    </Slide>
    <Slide index={1}>
      <Img />
    </Slide>
    <Slide index={2}>
      <Img />
    </Slide>
  </Carousel>

  <DotsGroup />
  <PrevButton />
  <NextButton />
</CarouselProvider>
```
