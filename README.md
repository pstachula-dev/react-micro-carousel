# Simple Headless Carousel React

### How to use:

```
<CarouselProvider width={500} total={3}>
  <Carousel>
    <Slide>
      <Img />
    </Slide>
    <Slide>
      <Img />
    </Slide>
    <Slide>
      <Img />
    </Slide>
  </Carousel>

  <DotsGroup />
  <PrevButton />
  <NextButton />
</CarouselProvider>
```
